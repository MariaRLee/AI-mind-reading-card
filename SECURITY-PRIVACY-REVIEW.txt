AI-Mind Care 情緒花卡：安全與隱私檢查摘要

已完成：
1. 移除所有分析、計數器與第三方追蹤程式。
2. 將 inline JavaScript 移至 app.js，以便套用較嚴格的 Content Security Policy。
3. 加入 CSP，僅允許同一網站的程式、樣式與圖片；禁止外部連線。
4. 將 innerHTML 改為安全的 textContent／DOM 節點建立方式。
5. 關閉情緒輸入欄的瀏覽器自動完成，降低本機留下輸入建議的機會。
6. 加入 no-referrer，點選 Home 時不傳送來源頁面資訊。
7. 加入清楚的網頁隱私說明。
8. 花卡路徑固定為 cards/，避免未來由外部輸入任意圖片網址。

部署檢查：
- GitHub Pages Settings → Pages → 勾選 Enforce HTTPS。
- GitHub repository 若為 Public，所有 HTML、JavaScript、CSS 與花卡圖片都可被下載；請勿把密碼、API key、個資或機密演算法放在前端檔案。
- 上傳圖片前，建議移除 EXIF／作者／位置等中繼資料。
- 此頁面沒有登入系統，不能識別個別使用者。
